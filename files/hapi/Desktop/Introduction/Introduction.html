Welcome Desktop Developers

The Handpoint mPOS a.k.a headstart platform on PC provides an application programming interface simplifying the development of your payment solutions considerably.

The solution is based up on socket communication between your application and Handpoint server running in your premises or even on your desktop.

An alternative library approach is available for those developing JAVA based solutions.

Handpoint for PC SDK includes:

  • Binaries: Handpoint PC, Handpoint Server and Handpoint client.
  • Sample: A sample client with source code available showing you Handpoint.
  • Docs: Integration process guide, integration manual and Javadoc.
  • Jars: Java .jar files for development using library approach.

If your solution is implemented using the Java programming language we recommend using the headstart Library. The library provides core functionality, interaction with the PED and communication with the processing server.

The Handpoint socket interface is a popular choice. The Headstart package includes a server which runs on a PC as a service and exposes the Handpoint Library functionality over a socket connection, via JSON messages (as string or as an object).
