getDeviceLogs

Fetches the logs from the device and reports them to the deviceLogsReady event


Parameters

ParameterTypeValidationNotes
deviceDeviceOptionalThis is used if you want to send the operation to a specific device. Recommended if you are using multiple devices

Events invoked

deviceLogsReady
Invoked when hapi has finished downloading logs from the card readewr

Returns

Boolean
true if the operation was successfully sent to device

Code examples

//Downloads logs from device
    api.getDeviceLogs();
