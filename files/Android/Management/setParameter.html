setParameter

Changes values of certain parameters on the card reader. Those are: BluetoothName, BluetoothPass, SystemTimeout, ScreenTimeout, SignatureTimeout


Parameters

ParameterTypeValidationNotes
paramDeviceParameterRequiredThe parameter to change
valueStringRequiredNew value of the parameter
deviceDeviceOptionalThis is used if you want to send the operation to a specific device. Recommended if you are using multiple devices

Events invoked

None

Returns

Boolean
true if the operation was successfully sent to device

Code examples

//Changes the bluetooth name of card reader
    api.setParameter(DeviceParameter.BluetoothName, "OrangeCardReader");
