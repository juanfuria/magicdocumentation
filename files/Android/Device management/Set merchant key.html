setSharedSecret

Saves the shared secret for the card reader so operations will be accepted by it


Parameters

ParameterTypeValidationNotes
sharedSecretStringRequiredThe shared secret
deviceDeviceOptionalThis is used if you want to send the operation to a specific device. Recommended if you are using multiple devices

Events invoked

None

Returns

Boolean
true if the operation was successfully sent to device

Code examples

//Sets the shared secret
    api.setSharedSecret("0102030405060708091011121314151617181920212223242526272829303132");
