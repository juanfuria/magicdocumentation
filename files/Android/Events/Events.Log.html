Log

Log events are events that the integrator can implement to gain access to API logs.

deviceLogsReady

deviceLogsReady event gets called when the logs requested by a call to getDeviceLogs() are ready.

ParameterTypeNotes
logsStringString containing the current log
deviceDeviceThe device that is invoking the event 

@Override
public void deviceLogsReady(String s, Device device) {
Log.d(App, s);
}

onMessageLogged

onMessageLogged event gets called for all log messages that are being logged. This is only intended for debugging.

ParameterTypeNotes
levelLogLevelAn enum containing the log level
messageStringA String containing the current log message 

@Override
public void onMessageLogged(LogLevel logLevel, String s) {
Log.d(SDK, s);
}
